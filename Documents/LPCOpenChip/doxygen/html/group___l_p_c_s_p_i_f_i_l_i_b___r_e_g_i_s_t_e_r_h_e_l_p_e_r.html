SPIFI_FAM_NODE_T* spifi_REG_FAMILY_CommonCommandSet ( void  )

Family registration function.

Returns
A pointer to a persistent SPIFI_DEV_FAMILY_T initialized for family.
Note
This function constructs and returns a non-volitile SPIFI_DEV_FAMILY_T structure that contains family specific information needed to register family. This function MUST NOT be called directly and should only be passed to the registration function spifiRegisterFamily()
